Zadáni cvičení 2

V příloze najdete html soubor (navigacni-lista.html). 
K němu si připojte váš vlastní soubor stylů a pomocí css jazyka vytvořte rozbalovací navigační lištu dle obrázku v příloze. V html souboru neprovádějte žádné zásahy, kromě připojení CSS souboru.

Vzhled navigační lišty je vidět na přiloženém obrázku. (Lišta zobrazuje po načtení stránky pouze názvy kategorií, po najetí myši se příslušná kategorie podbarví modře a rozbalí se podnabídka obsahující druhou úroveň seznamu. I položky druhé úrovně se po najetí myší podbarví modře).

    • Sudé a liché položky druhé úrovně navigace jsou barevně odlišené.
    • Poslední položka druhé úrovně menu má dole zaoblené rohy. (Např vlastnost border-radius.)
    • Po najetí kurzoru se daná položka zvýrazní (světlé pozadí, tmavý text).
    • První položka první úrovně navigace je barevně odlišená (nepovinné pro odevzdání).
    • Dopište si třetí úroveň navigace (i do html souboru), rozbalovací opět do strany (nepovinné pro odevzdání)
